Project Methodology

Airport Resilience Analysis During Hurricane Milton

Summary

This project evaluated the recovery performance of five Florida airports affected by Hurricane Milton in October 2024. Using flight operations data from the Bureau of Transportation Statistics and hourly wind speeds from the Meteostat API, a composite resilience score was developed to compare operational recovery while accounting for storm severity.

Key Metrics Used

  • Days to Recovery: Number of days until operations normalized
  • % Flights Cancelled (Oct): Percent of cancelled flights across October 2024
  • Peak Wind Speed: Maximum hourly average wind speed per airport

Resilience Score Formula

The formula balances operational impact (delays, cancellations) with storm severity:

resilience_score = ( (days_to_recovery + (pct_cancelled * 10)) / ((peak_wind_speed_kmh ** 0.5) / 10) )

Lower scores represent better resilience ‚Äî faster recovery with fewer disruptions, normalized by peak storm intensity.

Python Implementation

Data Engineering

  • Staging Layer: Raw FAA and Meteostat data loaded via SQL views
  • Prep Layer: Cleaned, filtered, joined by airport and date
  • Mart Layer: Final analytical dataset with recovery and weather KPIs
