Airport Resilience in Milton

Understanding infrastructure resilience through real data & research

Project Overview

This project analyzes the resilience of airport infrastructure in Milton, focusing on disruptions, recovery patterns, and recommendations for robustness in aviation planning.

My Role

I led the data analysis and visualization efforts, sourcing aviation traffic data, modeling disruptions, and presenting actionable insights using Python and geospatial tools.

Key Results

  • Identified top 3 resilience gaps in airport operations
  • Built a predictive model with 85% accuracy
  • Presented findings to stakeholders with interactive visuals
